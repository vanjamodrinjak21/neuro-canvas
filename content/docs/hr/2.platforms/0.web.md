# Web platforma

Web aplikacija na [neuro-canvas.com](https://neuro-canvas.com) primarni je i najstabilniji način korištenja NeuroCanvasa. Radi u cijelosti u pregledniku bez potrebe za instalacijom.

## Podržani preglednici

NeuroCanvas podržava **posljednje dvije glavne verzije** sljedećih preglednika:

| Preglednik | Minimalna verzija | Status |
|---------|----------------|--------|
| Google Chrome | 120+ | U potpunosti podržan |
| Mozilla Firefox | 121+ | U potpunosti podržan |
| Safari (macOS) | 17+ | U potpunosti podržan |
| Microsoft Edge | 120+ | U potpunosti podržan |
| Safari (iOS) | 17+ | Podržan (optimizirano za dodir) |
| Chrome (Android) | 120+ | Podržan (optimizirano za dodir) |

NeuroCanvas zahtijeva preglednik s podrškom za ES2022, WebWorkers, IndexedDB i Canvas API. Internet Explorer nije podržan.

## Progresivna web aplikacija

NeuroCanvas se može instalirati kao PWA na podržanim platformama. Nakon instalacije ponaša se poput izvorne aplikacije s vlastitim prozorom, ikonom i unosom na traci zadataka.

Za instalaciju PWA-a:

1. Otvorite [neuro-canvas.com](https://neuro-canvas.com) u Chromeu ili Edgeu.
2. Kliknite ikonu za instalaciju u adresnoj traci (ili koristite izbornik preglednika).
3. Potvrdite upit za instalaciju.

Na iOS Safariju koristite **Dijeli > Dodaj na početni zaslon**.

Prednosti PWA-a:
- Otvara se u vlastitom prozoru bez sučelja preglednika
- Izvanmrežni pristup prethodno učitanim mentalnim mapama putem IndexedDB-a (Dexie)
- Sinkronizacija u pozadini kada se mrežna veza uspostavi
- Automatska ažuriranja pri svakoj novoj implementaciji

## Obrada umjetnom inteligencijom na webu

Web platforma izvodi AI funkcionalnosti (semantičke ugradnje, izračun sličnosti) unutar **Web Workera** koristeći `@huggingface/transformers`. Na taj način glavni nit ostaje responzivan dok model izvodi inferenciju u pozadini.

Model za ugradnje (`all-MiniLM-L6-v2`) preuzima se i predmemorira u pregledniku pri prvom korištenju. Svako sljedeće učitavanje je trenutačno.

::callout{type="info"}
Početno preuzimanje modela iznosi otprilike 23 MB. Nakon pohrane u predmemoriju, model se učitava iz IndexedDB-a za manje od 2 sekunde na većini hardvera.
::

## Savjeti za bolju izvedbu

### Hardversko ubrzanje

- **Omogućite GPU ubrzanje** u postavkama preglednika. Chrome: `chrome://flags/#enable-gpu-rasterization`. Time se znatno poboljšava iscrtavanje platna za velike mentalne mape.
- NeuroCanvas koristi D3.js za rasporede s upravljivom silom i HTML Canvas API za iscrtavanje. Oba imaju koristi od hardverski ubrzanog kompozitinga.

### Velike mentalne mape (100+ čvorova)

- **Onemogućite sloj semantičkog polja** kada vam nije potreban. Vizualizacija polja izračunava parnu sličnost između svih čvorova, što se skalira kvadratično.
- Koristite **uklopi u prikaz** (tipkovni prečac `F`) za centriranje vidnog polja umjesto ručnog pomicanja po velikim mapama.
- Zatvorite **ploču AI prijedloga** kada je aktivno ne koristite kako biste smanjili pozadinsku obradu.
- Razmotrite podjelu vrlo velikih mapa (500+ čvorova) na više povezanih mapa radi bolje izvedbe.

### Upravljanje memorijom

- Svaka otvorena kartica preglednika s mentalnom mapom troši memoriju za D3 simulaciju sila, podatke čvorova i predmemorirane ugradnje. Zatvorite nekorištene kartice.
- Ako kartica preglednika postane spora, osvježite stranicu. Podaci mape automatski se pohranjuju u IndexedDB i odmah se obnavljaju.

## Mobilni web

NeuroCanvas je u potpunosti responzivan i optimiziran za mobilne preglednike s vidnim poljima od **390px i širim**.

Značajke mobilnog weba:
- **Donja traka s karticama** za navigaciju (Početna, Pretraži, Mape, Postavke) zamjenjuje bočnu traku na zaslonima širim od 768px i manjim
- **Platno optimizirano za dodir** s pinch-to-zoom i pomicanjem s dva prsta
- **Plutajuće alatne trake** pri vrhu i dnu uređivača platna
- **Donji listovi** za svojstva čvora i AI prijedloge
- **Minimalne površine dodira od 44px** na svim interaktivnim elementima

::callout{type="warning"}
Mobilni web pruža iste osnovne funkcionalnosti kao i desktopski web, ali su neke interakcije (višestruki odabir povlačenjem, kontekstni izbornici desnim klikom) prilagođene unosu dodirom. Za najbolje mobilno iskustvo s izvornima gestama pogledajte platformu [Mobilna aplikacija (Capacitor)](/docs/platforms/mobile-capacitor).
::

## Pohrana podataka

Na web platformi, podaci mentalne mape pohranjuju se na dva mjesta:

1. **Poslužitelj (PostgreSQL)** — mjerodavni izvor za autentificirane korisnike. Mape se automatski sinkroniziraju.
2. **Preglednik (IndexedDB putem Dexiea)** — lokalna predmemorija za izvanmrežni pristup i trenutačno učitavanje.

Automatska pohrana aktivira se nekoliko sekundi nakon promjena. Ako izgubite mrežnu vezu, promjene se lokalno stavljaju u red i sinkroniziraju kad se veza obnovi.

| `Ctrl/Cmd + Shift + Z` | Ponovi |
| `Delete / Backspace` | Izbriši odabrano |
| `F` | Uklopi u prikaz |
| `Ctrl/Cmd + +/-` | Povećaj/smanji prikaz |
