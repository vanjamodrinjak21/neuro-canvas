# Mobilna aplikacija (Capacitor)

::callout{type="warning"}
Mobilna aplikacija trenutačno je u **beta verziji**. Interakcije dodirom i korisničko sučelje su funkcionalni, ali se još uvijek dorađuju. Neke AI funkcionalnosti ograničene su u usporedbi s web i desktopskim platformama. Prijavite probleme na GitHubu.
::

NeuroCanvas Mobile izgrađen je s [Capacitorom 6](https://capacitorjs.com/) kako bi pružio izvorno iOS i Android iskustvo iz iste Vue.js baze koda koja pokreće web aplikaciju.

## Podrška za platforme

| Platforma | Minimalna verzija | Status |
|----------|----------------|--------|
| iOS | 16.0+ | Beta |
| Android | API 26 (Android 8.0+) | Beta |

## Arhitektura

Capacitor omotava NeuroCanvas Nuxt 3 SPA unutar izvornog WebViewa, s pristupom izvornim API-jima uređaja putem mosta. Aplikacija se kompilira kao statički SPA (`nuxt generate`) i pakira u izvorni projekt.

Izvorna ljuska pruža:
- **Izvorni zaslon pokretanja** — tamni brendarani zaslon (#032221) s prikazom od 2 sekunde
- **Integracija statusne trake** — tamni stil koji odgovara temi aplikacije
- **Upravljanje tipkovnicom** — automatsko mijenjanje veličine tijela kada se pojavi meka tipkovnica
- **Umetanja sigurnog područja** — automatsko rukovanje umetanjima sadržaja na iOS-u (usjek, Dynamic Island)

## Platno optimizirano za dodir

Uređivač platna prerađen je za unos dodirom:

### Geste

| Gesta | Radnja |
|---------|--------|
| Jedan dodir | Odaberi čvor |
| Dvostruki dodir | Uredi oznaku čvora |
| Dugi pritisak | Otvori kontekstni izbornik |
| Povlačenje jednim prstom | Premjesti odabrani čvor |
| Pinch s dva prsta | Povećaj/smanji prikaz |
| Pomicanje s dva prsta | Pomakni platno |
| Dodir s dva prsta | Poništi zadnju radnju |

### Površine dodira

Svi interaktivni elementi imaju minimalnu površinu dodira od **44x44 točke** (Apple HIG) / **48x48 dp** (Material Design), uključujući gumbe alatne trake, ručke čvora i stavke izbornika.

## Mobilno korisničko sučelje

Mobilno sučelje znatno se razlikuje od desktopskog rasporeda:

### Donja traka s karticama

Na zaslonima širim od 768px i manjim, bočna traka desktopa zamjenjuje se donjom trakom s četiri kartice:

- **Početna** — nadzorna ploča s nedavnim mapama i brzim radnjama
- **Pretraži** — pretraživanje svih mapa i čvorova
- **Mape** — prikaz liste svih mapa (jedan stupac, ne rešetka)
- **Postavke** — postavke aplikacije i upravljanje računom

### Plutajuće alatne trake

U uređivaču platna:
- **Gornja alatna traka** — naslov mape, poništi/ponovi, gumb za dijeljenje
- **Donja alatna traka** — stvaranje čvora, način spajanja, AI alati
- Obje alatne trake plutaju iznad platna i skupljaju se pri pomicanju radi maksimalnog prostora platna

### Donji listovi

Ploče koje se na desktopu prikazuju kao bočne trake (svojstva čvora, AI prijedlozi) na mobilnim uređajima prikazuju se kao **donji listovi**:
- Prevucite gore da biste proširili na punu visinu
- Prevucite dolje da biste skupili ili zatvorili
- Dodirnite pozadinu da biste zatvorili

### Prikaz liste za mape

Stranica "Sve mape" koristi raspored s jednim stupcem umjesto rešetkastog rasporeda koji se koristi na desktopu. Svaki element liste prikazuje naslov mape, datum zadnje izmjene, broj čvorova i minijaturni pregled.

## Izvorne značajke uređaja

Capacitor pruža pristup izvornim API-jima putem dodataka:

| Značajka | Dodatak | Opis |
|---------|--------|-------------|
| Datotečni sustav | `@capacitor/filesystem` | Pohrani/učitaj izveze mapa u pohranu uređaja |
| Dijeljenje | `@capacitor/share` | Izvorni list za dijeljenje pri izvozu mapa |
| Haptika | `@capacitor/haptics` | Taktilne povratne informacije pri stvaranju, brisanju čvorova i uspostavljanju veza |
| Dijaloški okvir | `@capacitor/dialog` | Izvorni dijaloški okviri za upozorenja i potvrde |

### Haptične povratne informacije

Aplikacija koristi haptične povratne informacije za:
- **Lagane** — odabir čvora, dodiri gumba alatne trake
- **Srednje** — stvaranje čvora, uspostavljanje veze
- **Jake** — brisanje čvora, destruktivne radnje

## AI funkcionalnosti na mobilnim uređajima

### Što radi

- Vizualizacija semantičkog polja (izračunata u web workeru)
- Otkrivanje sličnosti čvorova
- AI-generirani prijedlozi za mentalne mape
- Analiza u ploči uvida

### Poznata ograničenja

- **Izračun ugradnji je sporiji** — mobilni uređaji imaju manje CPU kapaciteta od deskopova. Grupna ugradnja velikih mapa (50+ čvorova) može potrajati primjetno dulje.
- **Nema lokalne ONNX inferencije** — za razliku od Tauri desktopske aplikacije, mobilna aplikacija izvodi ugradnje u JavaScript Web Workeru koristeći `@huggingface/transformers` (WASM). Na mobilnim uređajima nema izvornog Rust ML pozadinskog sustava.
- **Ograničenja memorije** — na uređajima s 4GB RAM-a ili manje, vrlo velike mape (200+ čvorova) s uključenim slojem semantičkog polja mogu uzrokovati ponovno učitavanje WebViewa. Onemogućite sloj semantičkog polja za bolju stabilnost na uređajima s malo memorije.
- **Pozadinska obrada** — iOS agresivno suspendira pozadinske WebViewove. Dugotrajne AI operacije mogu biti prekinute ako se aplikacija premjesti u pozadinu.

## Instalacija

### App Store (uskoro)

Izvorna izdanja za iOS (App Store) i Android (Google Play) planirana su za javno objavljivanje nakon završetka beta perioda. Pridružite se programu beta testiranja za rani pristup.

### Razvojna izdanja

Za lokalnu izgradnju i pokretanje mobilne aplikacije:

**Preduvjeti:**
- Node.js 20+
- Za iOS: macOS s Xcodeom 15+, CocoaPods
- Za Android: Android Studio s SDK 26+

```bash
# Install dependencies
npm install

# Generate the static SPA
npm run generate

# Sync native projects
npm run cap:sync

# Run on iOS simulator or device
npm run cap:ios

# Run on Android emulator or device
npm run cap:android
```

### iOS razvoj

```bash
# Open the iOS project in Xcode
npx cap open ios
```

iOS projekt koristi naziv sheme `NeuroCanvas` s automatskim upravljanjem umetanjima sadržaja.

### Android razvoj

```bash
# Open the Android project in Android Studio
npx cap open android
```

Android projekt cilja API 26+ s onemogućenim miješanim sadržajem radi sigurnosti i omogućenim otklanjanjem pogrešaka web sadržaja za razvojna izdanja.

## Konfiguracija

Konfiguracija Capacitora (`capacitor.config.ts`) definira postavke specifične za platformu:

- **ID aplikacije:** `com.neurocanvas.app`
- **Web direktorij:** `.output/public` (Nuxt statički izlaz)
- **iOS boja pozadine:** `#032221`
- **Android boja pozadine:** `#032221`
- **Zaslon pokretanja:** 2-sekundni brendirani zaslon s automatskim skrivanjem

Za razvoj s brzim osvježenjem, odkomentirajte opciju `server.url` u `capacitor.config.ts` i usmjerite je na vaš lokalni razvojni poslužitelj:

```typescript
server: {
  url: 'http://YOUR_LOCAL_IP:3000',
  cleartext: true
}
```

::callout{type="info"}
Koristite IP adresu lokalne mreže vašeg računala (ne `localhost`) kako bi mobilni uređaj ili emulator mogao dosegnuti razvojni poslužitelj putem mreže.
::
