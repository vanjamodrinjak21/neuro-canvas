# Desktop (Tauri)

::callout{type="warning"}
Desktopska aplikacija trenutačno je u **beta verziji**. Neke funkcionalnosti mogu biti nestabilne ili nepotpune. Gubitak podataka je malo vjerojatan, ali nije nemoguć — uvijek držite sinkronizaciju s oblakom uključenu kao sigurnosnu kopiju. Prijavite probleme na GitHubu.
::

NeuroCanvas Desktop izvorna je aplikacija izgrađena s [Taurijem 2](https://v2.tauri.app/) i Rust pozadinskim sustavom. Pruža izvornu izvedbu, lokalnu ML inferenciju i potpunu izvanmrežnu podršku.

## Zašto Tauri

Tauri pakira sistemski webview umjesto cijelog engine preglednika (kao što to čini Electron). Rezultat je:

- **Manji binarni paket** — instalacijska datoteka NeuroCanvasa tipično je 15-25 MB ovisno o platformi, u usporedbi s 150+ MB za aplikacije temeljene na Electronu
- **Manja potrošnja memorije** — koristi OS-izvorne webviewove (WebKit na macOSu, WebView2 na Windowsima, WebKitGTK na Linuxu)
- **Izvorni Rust pozadinski sustav** — CPU-intenzivne operacije (ML inferencija, datotečni U/I) izvode se u izvornom Rustu, ne u JavaScriptu

## Instalacija

### macOS

1. Preuzmite `.dmg` datoteku za svoju arhitekturu s [stranice izdanja](https://github.com/neurocanvas/neurocanvas/releases).
2. Otvorite `.dmg` i povucite **NeuroCanvas** u mapu **Aplikacije**.
3. Prilikom prvog pokretanja macOS može blokirati aplikaciju. Idite na **Postavke sustava > Privatnost i sigurnost** i kliknite **Svejedno otvori**.

Ciljani paketi:
- `aarch64-apple-darwin` — Apple Silicon (M1/M2/M3/M4)
- `x86_64-apple-darwin` — Intel
- `universal-apple-darwin` — Univerzalni binarni paket (obje arhitekture)

### Windows

1. Preuzmite `.msi` ili `.exe` instalacijsku datoteku sa stranice izdanja.
2. Pokrenite instalacijsku datoteku i slijedite upute.
3. Windows SmartScreen može prikazati upozorenje za nepotpisana izdanja. Kliknite **Više informacija > Svejedno pokreni**.

Windows zahtijeva **WebView2** runtime (uključen u Windows 11, automatski se instalira na Windowsima 10).

### Linux

**AppImage (preporučeno):**
1. Preuzmite `.AppImage` datoteku sa stranice izdanja.
2. Učinite je izvršnom: `chmod +x NeuroCanvas_*.AppImage`
3. Pokrenite je: `./NeuroCanvas_*.AppImage`

**Debian/Ubuntu (.deb):**
```bash
sudo dpkg -i neurocanvas_*.deb
sudo apt-get install -f  # Install any missing dependencies
```

Linux zahtijeva WebKitGTK 4.1+. Na Ubuntuu 22.04+:
```bash
sudo apt install libwebkit2gtk-4.1-0
```

## Lokalna ML inferencija

Desktopska aplikacija uključuje izvornu integraciju ONNX Runtimea za lokalno izvođenje ML modela. To je glavna prednost u odnosu na web platformu.

### Kako to funkcionira

Rust pozadinski sustav učitava model `all-MiniLM-L6-v2` za rečenične transformere putem [ort](https://github.com/pykeio/ort) paketa. Inferencija se izvodi izvorički na CPU-u (ili Apple Neural Engineu / CoreMLu na macOSu) bez prolaska kroz JavaScript.

Dostupne ML naredbe izložene frontendu:

| Naredba | Opis |
|---------|-------------|
| `ml_init` | Učitaj ONNX model (kvantiziran ili pune preciznosti) |
| `ml_embed` | Generiraj jednu tekstualnu ugradnju (384 dimenzije) |
| `ml_embed_batch` | Grupno ugradi više tekstova s događajima napretka |
| `ml_compute_similarities` | Izračunaj parne kosinusne sličnosti iznad praga |
| `ml_download_full_model` | Preuzmi model pune preciznosti s HuggingFacea |
| `ml_is_full_model_available` | Provjeri je li model pune preciznosti preuzet |
| `ml_delete_full_model` | Izbriši preuzeti model pune preciznosti |

### Varijante modela

- **Kvantiziran** (zadano) — pakiran uz aplikaciju (~30 MB). Brža inferencija, nešto niža točnost. Učitava se pri pokretanju.
- **Puna preciznost** — preuzima se na zahtjev (~90 MB). Veća točnost, sporija inferencija. Pohranjuje se u direktorij podataka aplikacije.

Na Apple Silicon Macovima, ONNX runtime koristi **CoreML** ubrzanje za bržu inferenciju.

### Usporedba izvedbe

| Operacija | Web (WASM) | Desktop (Rust/ONNX) |
|-----------|-----------|---------------------|
| Jedna ugradnja | ~50ms | ~8ms |
| Grupno (100 čvorova) | ~3s | ~400ms |
| Sličnost (100 čvorova) | ~200ms | ~30ms |

*Mjerila na Apple M2, 16GB RAM. Stvarna izvedba ovisi o hardveru.*

## Izvanmrežni način rada

Desktopska aplikacija u potpunosti radi izvanmrežno za lokalne mape:

- Mape se pohranjuju lokalno u IndexedDB unutar webviewa
- ML inferencija izvodi se u cijelosti na uređaju — nisu potrebni API pozivi
- Vizualizacija semantičkog polja, izračun sličnosti i AI prijedlozi rade izvanmrežno
- Autentifikacija se zaobilazi u desktopskom načinu rada (aplikacija provjerava postoji li `window.__TAURI__`)

Kada je dostupna internetska veza, mape se automatski sinkroniziraju s poslužiteljem. Izvanmrežne promjene stavljaju se u red i spajaju pri ponovnom povezivanju.

::callout{type="info"}
AI funkcionalnosti koje zahtijevaju vanjske LLM API-je (generiranje mapa iz opisa, napredna analiza uvida) i dalje zahtijevaju internetsku vezu i konfigurirane API ključeve.
::

## Značajke specifične za desktop

### Izvorni dijalozi za datoteke

Desktopska aplikacija koristi izvorne OS dijaloge za datoteke pri:
- Izvozu mapa u formatu JSON, PNG ili SVG
- Uvozu podataka mapa iz datoteka
- Odabiru prilagođenih datoteka modela

Ovo je omogućeno pomoću dodataka `tauri-plugin-dialog` i `tauri-plugin-fs`.

### Integracija sa sustavom

- **Upravljanje prozorima** — prozor s promjenjivom veličinom (minimum 800x600), pamti veličinu i položaj
- **Izvorni izbornici** — standardna traka izbornika OS-a na macOSu
- **Duboke veze** — otvarajte `neurocanvas://` URL-ove izravno u aplikaciji
- **Automatsko ažuriranje** — provjerava ažuriranja pri pokretanju (kada je mrežno spojen)

### Otkrivanje hardvera

Aplikacija otkriva hardverske mogućnosti pri pokretanju radi optimizacije ML izvedbe:

- Otkrivanje Apple Silicona (za CoreML ubrzanje)
- Otkrivanje diskretnog GPU-a (AMD/NVIDIA na macOSu)
- Rezultati određuju koja se varijanta modela preporučuje

## Izgradnja iz izvornog koda

Preduvjeti:
- Node.js 20+
- Rust 1.77+
- Ovisnosti specifične za platformu (pogledajte [Tauri preduvjete](https://v2.tauri.app/start/prerequisites/))

```bash
# Clone the repository
git clone https://github.com/neurocanvas/neurocanvas.git
cd neurocanvas

# Install dependencies
npm install

# Run in development mode
npm run tauri:dev

# Build for production
npm run tauri:build

# Platform-specific builds
npm run tauri:build:mac-arm       # Apple Silicon
npm run tauri:build:mac-x86       # Intel Mac
npm run tauri:build:mac-universal # Universal binary
npm run tauri:build:win-x86       # Windows x64
npm run tauri:build:win-arm       # Windows ARM
```

Izgrađena aplikacija ispisuje se u `src-tauri/target/release/bundle/`.
