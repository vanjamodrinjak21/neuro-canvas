# AI krajnje točke

NeuroCanvas pruža AI funkcionalnosti putem sloja proxyja na strani poslužitelja. Proxy prosljeđuje zahtjeve konfiguriranome AI pružatelju korisnika (OpenAI, Anthropic, OpenRouter, Ollama ili prilagođena krajnja točka kompatibilna s OpenAIem), izbjegavajući CORS probleme i zadržavajući API ključeve na strani poslužitelja.

AI krajnje točke koriste API ključeve iz trezora vjerodajnica na strani poslužitelja. Ključevi su šifrirani AES-GCM algoritmom i dešifriraju se samo pri prosljeđivanju zahtjeva AI pružatelju. Klijent specificira koji pružatelj se koristi, a poslužitelj dohvaća odgovarajući ključ iz trezora.

## Podržani pružatelji

| Pružatelj | Zadani osnovni URL | Zadani model |
|---|---|---|
| `openai` | `https://api.openai.com/v1` | `gpt-4o-mini` |
| `anthropic` | `https://api.anthropic.com/v1` | `claude-sonnet-4-20250514` |
| `openrouter` | `https://openrouter.ai/api/v1` | `openai/gpt-4o-mini` |
| `ollama` | `http://localhost:11434` | `llama3.2` |
| `custom` | Korisnik definira (obavezno) | `default` |

Svi pružatelji prihvaćaju neobavezno nadjačavanje `baseUrl` za vlastito postavljene ili alternativne krajnje točke.

## Dovršavanja

Generirajte AI odgovor bez strujanja.

```
POST /api/ai/completions
Content-Type: application/json

{
  "provider": "openai",
  "model": "gpt-4o-mini",
  "messages": [
    { "role": "user", "content": "Suggest 3 subtopics for 'Machine Learning'" }
  ],
  "systemPrompt": "You are a mind mapping assistant. Respond with structured JSON.",
  "maxTokens": 500,
  "temperature": 0.7
}
```

Poslužitelj dohvaća API ključ iz trezora vjerodajnica na temelju polja `provider`. U tijelu zahtjeva ne šalje se API ključ.

**Parametri:**

| Polje | Vrsta | Obavezno | Zadano | Opis |
|---|---|---|---|---|
| `provider` | `string` | Da | | Identifikator pružatelja (vidi gornju tablicu) |
| `model` | `string` | Ne | Zadano pružatelja | Identifikator modela |
| `messages` | `array` | Da | | Poruke razgovora u formatu `[{ role, content }]` |
| `systemPrompt` | `string` | Ne | | Sistemska poruka dodana ispred razgovora |
| `maxTokens` | `number` | Ne | `500` | Maksimalni broj tokena u odgovoru |
| `temperature` | `number` | Ne | `0.7` | Temperatura uzorkovanja (0.0 - 2.0) |
| `baseUrl` | `string` | Ne | Zadano pružatelja | Nadjačaj osnovni URL pružatelja |

**Odgovor:**
```json
{
  "content": "Here are 3 subtopics for Machine Learning:\n1. Supervised Learning\n2. Neural Networks\n3. Reinforcement Learning",
  "usage": {
    "prompt_tokens": 42,
    "completion_tokens": 38,
    "total_tokens": 80
  }
}
```

Polje `usage` je `null` za Ollama odgovore koji ne izvještavaju o upotrebi tokena.

**Greške:**

| Status | Uvjet |
|---|---|
| `400` | Nedostaje `provider` ili u trezoru nije pronađena vjerodajnica |
| `400` | Prilagođenom pružatelju nedostaje `baseUrl` |
| `400` | Nepodržani pružatelj |
| Upstream status | Proslijeđeno od AI pružatelja (npr. 401 za nevaljani ključ, 429 za ograničenje brzine) |
| `500` | Neočekivani kvar proxyja |

## Dovršavanja strujanjem

Generirajte AI odgovor strujanjem isporučen kao Server-Sent Events (SSE).

```
POST /api/ai/stream
Content-Type: application/json

{
  "provider": "anthropic",
  "model": "claude-sonnet-4-20250514",
  "messages": [
    { "role": "user", "content": "Explain quantum computing in simple terms" }
  ],
  "systemPrompt": "You are a helpful mind mapping assistant.",
  "maxTokens": 500,
  "temperature": 0.7
}
```

Tijelo zahtjeva identično je krajnjoj točki za dovršavanja.

**Zaglavlja odgovora:**
```
Content-Type: text/event-stream
Cache-Control: no-cache
Connection: keep-alive
```

**Format SSE događaja:**

Svaki događaj je JSON objekt na liniji `data:`. Emitiraju se tri vrste događaja:

**Delta (isječak sadržaja):**
```
data: {"type":"delta","content":"Quantum computing uses"}

data: {"type":"delta","content":" quantum bits, or qubits,"}
```

**Done (tok završen):**
```
data: {"type":"done","usage":{"prompt_tokens":42,"completion_tokens":120,"total_tokens":162}}
```

**Error (greška):**
```
data: {"type":"error","message":"API error: 429"}
```

Polje `usage` u događaju `done` je `null` kada pružatelj ne izvještava o upotrebi (Ollama) ili kada upstream pružatelj šalje upotrebu odvojeno.

### Konzumiranje na strani klijenta

```typescript
const response = await fetch('/api/ai/stream', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ provider, model, messages })
})

const reader = response.body.getReader()
const decoder = new TextDecoder()

while (true) {
  const { done, value } = await reader.read()
  if (done) break

  const text = decoder.decode(value, { stream: true })
  const lines = text.split('\n')

  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const event = JSON.parse(line.slice(6))
      if (event.type === 'delta') {
        // Append event.content to the UI
      } else if (event.type === 'done') {
        // Stream complete
      } else if (event.type === 'error') {
        // Handle error
      }
    }
  }
}
```

## Predmemorija AI odgovora

NeuroCanvas predmemorira AI odgovore u Redisu radi izbjegavanja redundantnih API poziva za identične upite.

### Dohvaćanje predmemoriranog odgovora

```
GET /api/ai/cache?hash=abc123
```

**Odgovor (pogodak predmemorije):**
```json
{
  "hit": true,
  "response": "Cached AI response text...",
  "usage": { "prompt_tokens": 42, "completion_tokens": 38 },
  "cachedAt": 1718444400000
}
```

**Odgovor (promašaj predmemorije):**
```json
{
  "hit": false,
  "response": null,
  "usage": null
}
```

### Pohrana odgovora u predmemoriju

```
POST /api/ai/cache
Content-Type: application/json

{
  "hash": "abc123",
  "response": "The AI response text to cache",
  "usage": { "prompt_tokens": 42, "completion_tokens": 38 },
  "ttlSeconds": 3600
}
```

**Parametri:**

| Polje | Vrsta | Obavezno | Zadano | Opis |
|---|---|---|---|---|
| `hash` | `string` | Da | | Deterministički hash upita |
| `response` | `string` | Da | | Tekst AI odgovora |
| `usage` | `object` | Ne | `null` | Metapodaci o upotrebi tokena |
| `ttlSeconds` | `number` | Ne | `3600` | TTL predmemorije u sekundama (zadano 1 sat) |

**Odgovor:**
```json
{
  "success": true
}
```

## AI korisnička memorija

Trajno pohranjujte i dohvaćajte AI preferencije i kontekst specifične za korisnika u Redisu.

### Dohvaćanje memorije

```
GET /api/ai/memory?userId=clx1abc2d0001
```

**Odgovor:**
```json
{
  "data": {
    "preferredStyle": "concise",
    "domainContext": "software-engineering",
    "lastTopics": ["machine-learning", "system-design"]
  }
}
```

Vraća `{ "data": null }` ako nema pohranjene memorije.

### Pohrana memorije

```
POST /api/ai/memory
Content-Type: application/json

{
  "userId": "clx1abc2d0001",
  "data": {
    "preferredStyle": "concise",
    "domainContext": "software-engineering",
    "lastTopics": ["machine-learning", "system-design"]
  }
}
```

Unosi memorije imaju TTL od 30 dana i automatski se čiste.

**Odgovor:**
```json
{
  "success": true
}
```

## Testiranje veze

Testirajte vezu s AI pružateljem i po želji dohvatite dostupne modele.

```
POST /api/ai/test-connection
Content-Type: application/json

{
  "provider": "openai",
  "baseUrl": "https://api.openai.com/v1"
}
```

**Odgovor (uspjeh):**
```json
{
  "success": true,
  "message": "API key verified"
}
```

**Odgovor (uspjeh s modelima — Ollama, OpenRouter):**
```json
{
  "success": true,
  "message": "Connected successfully",
  "models": [
    { "id": "llama3.2", "name": "llama3.2" },
    { "id": "mistral", "name": "mistral" }
  ]
}
```

**Odgovor (neuspjeh):**
```json
{
  "success": false,
  "message": "Invalid API key"
}
```

### Ponašanje specifično za pružatelja

| Pružatelj | Metoda testiranja | Vraća modele |
|---|---|---|
| `openai` | `GET /models` s Bearer autentifikacijom | Ne |
| `anthropic` | Šalje minimalni zahtjev za dovršavanje (1 token) | Ne |
| `openrouter` | `GET /models` s Bearer autentifikacijom | Da (do 50) |
| `ollama` | `GET /api/tags` | Da (svi lokalni modeli) |
| `custom` | `HEAD` zahtjev na osnovni URL | Ne |

Za Anthropic, odgovor `429 Too Many Requests` tretira se kao uspješan test veze (ključ je valjan, samo je ograničena brzina).

## Ugradnje

Pohranite i dohvatite semantičke ugradnje za čvorove mentalne mape. Ugradnje se generiraju na strani klijenta (putem transformers.js ili Tauri ML mosta) i predmemoriraju na strani poslužitelja u Redisu za brzo dohvaćanje.

### Pohrana ugradnji

```
POST /api/embeddings
Content-Type: application/json

{
  "mapId": "clx1map001",
  "embeddings": [
    {
      "nodeId": "node_abc",
      "text": "Machine Learning",
      "embedding": [0.012, -0.034, 0.056, ...]
    },
    {
      "nodeId": "node_def",
      "text": "Neural Networks",
      "embedding": [-0.023, 0.045, 0.078, ...]
    }
  ]
}
```

Zahtijeva autentifikaciju. Svaka ugradnja pohranjuje se s dva Redis ključa:
1. **Po hash vrijednosti teksta** — za dohvaćanje adresabilno sadržajem kroz mape
2. **Po ID-u čvora unutar mape** — za dohvaćanje specifično za mapu

Ugradnje imaju TTL od 7 dana.

**Odgovor:**
```json
{
  "success": true,
  "stored": 2
}
```

Unosi koji nemaju `nodeId`, `text` ili `embedding` tiho se preskakuju.

### Dohvaćanje ugradnji

```
GET /api/embeddings?mapId=clx1map001&nodeIds=node_abc,node_def,node_ghi
```

**Parametri upita:**

| Parametar | Vrsta | Opis |
|---|---|---|
| `mapId` | `string` | ID mape za ograničavanje pretraživanja |
| `nodeIds` | `string` | Lista ID-ova čvorova odvojena zarezima |

**Odgovor:**
```json
{
  "embeddings": [
    { "nodeId": "node_abc", "embedding": [0.012, -0.034, 0.056, ...] },
    { "nodeId": "node_def", "embedding": [-0.023, 0.045, 0.078, ...] }
  ],
  "found": 2,
  "requested": 3
}
```

Čvorovi bez predmemoriranih ugradnji izostavljaju se iz odgovora. Usporedite `found` s `requested` kako biste odredili koji čvorovi trebaju generiranje ugradnji.

### Brisanje ugradnji mape

Uklonite sve predmemorirane ugradnje za određenu mapu.

```
DELETE /api/embeddings/:mapId
```

**Odgovor:**
```json
{
  "success": true,
  "mapId": "clx1map001"
}
```

Koristi Redis podudaranje uzoraka (`emb:map:{mapId}:*`) za brisanje svih ugradnji čvorova mape u jednoj operaciji.
