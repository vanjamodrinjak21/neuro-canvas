# Pregled API-ja

NeuroCanvas API je RESTful JSON API izgrađen na Nuxt 3 serverskim rutama. Sve krajnje točke nalaze se pod `/api` i slijede dosljedne konvencije za autentifikaciju, rukovanje greškama i strukturu odgovora.

## Osnovni URL

| Okruženje | Osnovni URL |
|---|---|
| Produkcija | `https://neuro-canvas.com/api` |
| Lokalni razvoj | `http://localhost:3000/api` |
| Vlastito postavljanje | `https://your-domain.com/api` |

Svi zahtjevi i odgovori koriste `Content-Type: application/json` osim ako nije drugačije navedeno (krajnje točke za strujanje koriste `text/event-stream`).

## Autentifikacija

Većina krajnjih točaka zahtijeva autentificiranu sesiju. NeuroCanvas koristi [NextAuth.js](https://next-auth.js.org/) (putem `@sidebase/nuxt-auth`) sa strategijom sesija temeljenom na JWT-u.

### Dobivanje tokena sesije

Nakon prijave putem jednog od dostupnih pružatelja (Google, GitHub, e-pošta ili Vjerodajnice), poslužitelj postavlja sigurni HTTP-only kolačić sesije. Zahtjevi iz preglednika automatski uključuju ovaj kolačić. Za korištenje web aplikacije nije potreban ručni zaglavlje `Authorization`.

### Provjera valjanosti sesije

Svaka zaštićena krajnja točka provjerava sesiju na strani poslužitelja. Ako sesija nedostaje ili je istekla, poslužitelj vraća odgovor `401 Unauthorized`. Sesije imaju konfigurabilno trajanje i automatski se obnavljaju za vrijeme aktivnog korištenja.

## Ograničavanje brzine

Sve osjetljive API krajnje točke su ograničene brzinom radi sprječavanja zlouporabe. Ograničenja brzine primjenjuju se po korisniku ili po IP adresi ovisno o krajnjoj točki. Sljedeće operacije su ograničene brzinom:

- Operacije sinkronizacije
- Registracija
- Promjene lozinke
- Brisanje računa
- Izvoz podataka
- Postavljanje i provjera TOTP-a
- AI krajnje točke

Kada se premaši ograničenje brzine, poslužitelj vraća `429 Too Many Requests`.

## Format odgovora

### Uspješni odgovori

Odgovori NeuroCanvas API-ja su ravni JSON objekti. Oblik ovisi o krajnjoj točki, ali uobičajeni obrasci uključuju:

**Jedan resurs:**
```json
{
  "id": "clx1abc2d0001",
  "email": "user@example.com",
  "name": "Jane Doe",
  "createdAt": "2026-01-15T10:30:00.000Z"
}
```

**Kolekcija s metapodacima:**
```json
{
  "maps": [
    { "id": "map_1", "title": "Project Roadmap", "syncVersion": 3 }
  ],
  "deleted": ["map_old_1"]
}
```

**Rezultat operacije:**
```json
{
  "success": true,
  "stored": 12
}
```

### Straničeni odgovori

Krajnje točke za povlačenje pri sinkronizaciji podržavaju vremensku paginaciju pomoću parametra upita `since` (ISO 8601 vremenska oznaka). Odgovor uključuje sve zapise ažurirane nakon navedenog vremena:

```
GET /api/sync/pull?since=2026-01-01T00:00:00.000Z
```

### Odgovori s greškama

Sve greške slijede Nuxt/H3 format grešaka s HTTP statusnim kodom i čitljivom porukom:

```json
{
  "statusCode": 400,
  "statusMessage": "Missing required fields: mapId, data, title, checksum"
}
```

AI proxy krajnje točke vraćaju greške s poljem `message` umjesto toga:

```json
{
  "statusCode": 500,
  "message": "AI request failed"
}
```

## HTTP statusni kodovi

| Kod | Značenje | Kada se koristi |
|---|---|---|
| `200` | OK | Uspješan GET, PATCH, POST ili DELETE |
| `201` | Created | Trenutačno se ne koristi (uspješno stvaranje vraća 200) |
| `400` | Bad Request | Nedostajući ili nevaljani parametri, greška provjere valjanosti |
| `401` | Unauthorized | Nedostajuća ili istekla sesija |
| `403` | Forbidden | Valjana sesija, ali nedovoljne ovlasti |
| `404` | Not Found | Resurs ne postoji |
| `409` | Conflict | Konflikt verzije sinkronizacije (poslužitelj ima noviju verziju) ili duplicirani resurs |
| `423` | Locked | Mapa je zaključana drugom operacijom sinkronizacije |
| `429` | Too Many Requests | Prekoračeno ograničenje brzine |
| `500` | Internal Server Error | Neočekivani kvar poslužitelja |

## CSRF zaštita

NeuroCanvas primjenjuje višestruke slojeve CSRF zaštite na svim krajnjim točkama koje mijenjaju stanje. To uključuje CSRF tokene temeljene na kolačićima i provjeru prilagođenih zaglavlja. Međudomenski zahtjevi obrazaca su blokirani.

## Provjera ispravnosti

Javna krajnja točka za provjeru ispravnosti dostupna je bez autentifikacije:

```
GET /api/health
```

```json
{
  "status": "ok",
  "timestamp": "2026-01-15T10:30:00.000Z",
  "version": "1.0.0"
}
```

## Predmemoriranje

NeuroCanvas koristi predmemoriranje na strani poslužitelja za podatke mapa, AI odgovore, ugradnje i stanje sinkronizacije. Predmemorija se automatski poništava pri operacijama pisanja (push, ažuriranje, brisanje).
