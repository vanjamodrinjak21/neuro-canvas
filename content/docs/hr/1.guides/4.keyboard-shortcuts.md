# Tipkovnički prečaci

NeuroCanvas podržava tipkovničke prečace za gotovo svaku uobičajenu radnju. Njihovo savladavanje značajno ubrzava vaš tijek rada.

Pritisnite `?` ili otvorite modal s prečacima iz izbornika za pomoć da biste vidjeli ovaj popis unutar aplikacije.

## Navigacija

| Radnja | Prečac |
|--------|--------|
| Pomicanje platna | `Space` + povlačenje |
| Pomicanje (tipke sa strelicama) | `Strelica lijevo` / `desno` / `gore` / `dolje` |
| Povećanje | `Cmd/Ctrl` + `+` |
| Smanjenje | `Cmd/Ctrl` + `-` |
| Prilagodi sve čvorove zaslonu | `F` |
| Vrati na 100% zum | `1` |

## Alati

| Radnja | Prečac |
|--------|--------|
| Alat za odabir | `V` |
| Alat za pomicanje (ruka) | `H` |
| Alat za dodavanje čvora | `N` |
| Alat za povezivanje | `C` |
| Ploča predložaka | `T` |

## Uređivanje

| Radnja | Prečac |
|--------|--------|
| Uredi tekst čvora | Dvostruki klik na čvor |
| Izbriši odabrano | `Delete` ili `Backspace` |
| Dupliciraj odabrano | `Cmd/Ctrl` + `D` |
| Odaberi sve | `Cmd/Ctrl` + `A` |
| Poništi | `Cmd/Ctrl` + `Z` |
| Ponovi | `Cmd/Ctrl` + `Shift` + `Z` |

## Graf i veze

| Radnja | Prečac |
|--------|--------|
| Otvori prikaz grafa | `Cmd/Ctrl` + `G` |
| Uključi/isključi ploču povratnih veza | `Cmd/Ctrl` + `B` |
| Uključi/isključi semantičko polje | `S` |

## AI značajke

| Radnja | Prečac |
|--------|--------|
| Pametno proširenje (AI prijedlozi) | `Cmd/Ctrl` + `E` |
| Prihvati gornji prijedlog | `Cmd/Ctrl` + `Enter` |

## Miš i podloga za praćenje

| Radnja | Unos |
|--------|------|
| Odabir čvora | Klik |
| Višestruki odabir | `Shift` + klik |
| Odabir okvirom | Klik + povlačenje na praznom platnu |
| Pomicanje platna | Srednja tipka miša + povlačenje |
| Zum | Kotačić za pomicanje |
| Kontekstni izbornik | Desni klik |
| Pomicanje čvora | Klik + povlačenje na čvoru |
| Stvaranje veze | Alat za povezivanje (`C`) + povlačenje između čvorova |

## Dodirne geste (mobilni uređaji / podloga za praćenje)

| Radnja | Gesta |
|--------|-------|
| Pomicanje | Povlačenje s dva prsta |
| Zum | Štipanje prema unutra / prema van |
| Odabir | Dodir |
| Uređivanje čvora | Dvostruki dodir |
| Kontekstni izbornik | Dugi pritisak |

## Savjeti

- **Trajnost alata**: Nakon pritiska `N` (alat za čvorove) ili `C` (alat za povezivanje), alat ostaje aktivan dok ne pritisnete `V` (odabir) ili `Escape`. To vam omogućuje postavljanje više čvorova ili crtanje više veza uzastopce bez ponovnog aktiviranja alata.
- **Brze izmjene**: Najbrži način dodavanja i označavanja čvora je: `N` (aktiviraj alat za čvorove), klik (postavi čvor), odmah počni tipkati (čvor ulazi u način uređivanja pri izradi).
- **Istraživanje pomoću tipkovnice**: Koristite `Cmd/Ctrl + G` za otvaranje prikaza grafa radi strukturalnog pregleda, zatim `F` da biste prilagodili sve zaslonu kad se vratite na platno.
