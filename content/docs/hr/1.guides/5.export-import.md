# Izvoz i uvoz

NeuroCanvas podržava više formata izvoza kako biste mogli napraviti sigurnosnu kopiju svog rada, podijeliti ga s drugima ili integrirati u druge alate.

## Formati izvoza

### JSON (potpuni izvoz podataka)

JSON izvoz daje potpunu, strojno čitljivu reprezentaciju vaše mape -- svaki čvor, vezu, poziciju, stil, opis i dio metapodataka.

**Kada ga koristiti:**
- Sigurnosno kopiranje mape radi zaštite.
- Prijenos mape između uređaja (npr. s desktop aplikacije na web).
- Dijeljenje mape s drugim NeuroCanvas korisnikom koji je želi uređivati.
- Kontrola verzija (pohranite JSON datoteke u Git za praćenje razvoja mape kroz vrijeme).

**Kako izvesti:**
1. Otvorite mapu koju želite izvesti.
2. Otvorite izbornik s više mogućnosti (ikona tri točke) u gornjoj traci.
3. Odaberite **Izvezi kao JSON**.
4. Preuzima se `.json` datoteka s nazivom datoteke temeljenim na naslovu vaše mape.

**Što datoteka sadrži:**
- Metapodatke mape (naslov, opis, datum izrade, datum posljednje izmjene).
- Sve čvorove s njihovim pozicijama, veličinama, stilovima, sadržajem, opisima i metapodacima.
- Sve veze s njihovim referencama izvora/cilja, oznakama i stilovima.
- Identifikator korijenskog čvora.
- Semantičke podatke (ugradnje, predmemorija sličnosti) ako su dostupni.

### PNG (vizualna snimka)

PNG izvoz prikazuje trenutni prikaz platna kao rastersku sliku.

**Kada ga koristiti:**
- Ugrađivanje mape u prezentaciju, izvješće ili dokument.
- Dijeljenje brze vizualizacije s nekim tko ne koristi NeuroCanvas.
- Izrada sličice ili slike pregleda.

**Kako izvesti:**
1. Uredite prikaz platna da prikaže ono što želite snimiti. Koristite `F` (prilagodi zaslonu) za prikaz cijele mape.
2. Otvorite izbornik s više mogućnosti u gornjoj traci.
3. Odaberite **Izvezi kao PNG**.
4. Preuzima se `.png` datoteka.

::callout{type="info"}
PNG snima platno na trenutnoj razini zuma i vidokrugu. Za najbolji rezultat, prilagodite sve čvorove zaslonu prije izvoza kako ništa ne bi bilo odrezano.
::

### Markdown (tekstualni pregled)

Markdown izvoz pretvara hijerarhijsku strukturu vaše mape u uvučeni tekstualni pregled.

**Kada ga koristiti:**
- Lijepljenje sadržaja mape u dokument, e-poštu ili aplikaciju za bilješke.
- Izrada tekstualnog sažetka vizualne mape.
- Slanje strukture mape u druge alate koji prihvaćaju Markdown.

**Kako izvesti:**
1. Otvorite mapu.
2. Otvorite izbornik s više mogućnosti u gornjoj traci.
3. Odaberite **Izvezi kao Markdown**.
4. Preuzima se `.md` datoteka.

**Kako izgleda izlaz:**
Korijenski čvor postaje H1 naslov. Čvorovi prve razine postaju stavke popisa. Dublji čvorovi su više uvučeni. Opisi čvorova, kada su prisutni, uključeni su ispod svake stavke.

```markdown
# Machine Learning Fundamentals

- Supervised Learning
  - Classification
  - Regression
  - Decision Trees
- Unsupervised Learning
  - Clustering
  - Dimensionality Reduction
- Reinforcement Learning
  - Q-Learning
  - Policy Gradient Methods
```

### Dijeljenje kao tekst (mobilni uređaji)

Na mobilnim uređajima s podrškom za list za dijeljenje:

1. Otvorite mapu.
2. Dodirnite gumb za dijeljenje.
3. Odaberite **Podijeli kao tekst**.
4. Sistemski list za dijeljenje otvara se s Markdown pregledom spremnim za slanje u bilo koju aplikaciju.

Na desktop preglednicima, ovo kopira Markdown u vašu međuspremnik.

## Uvoz

### Uvoz iz JSON-a

Za vraćanje mape iz JSON izvoza:

1. Idite na **Nadzornu ploču**.
2. Kliknite **Uvezi**.
3. Odaberite `.json` datoteku iz vašeg datotečnog sustava.
4. NeuroCanvas raščlanjuje datoteku i stvara novu mapu sa svim izvornim podacima.
5. Uvezena mapa pojavljuje se na vašem popisu mapa i može se normalno otvoriti.

**Što se uvozi:**
- Svi čvorovi, veze i njihova svojstva.
- Pozicije i stilovi čvorova (raspored je sačuvan točno).
- Opisi i metapodaci.
- Oznaka korijenskog čvora.

**Što se ne prenosi:**
- Stanje sinkronizacije (uvezena mapa tretira se kao nova lokalna mapa).
- Izračunati podaci poput semantičkih ugradnji (oni se automatski regeneriraju).

### Rukovanje greškama pri uvozu

Ako je JSON datoteka neispravnog oblika ili nekompatibilna:

- NeuroCanvas provjerava strukturu datoteke prije uvoza.
- Ako nedostaju obavezna polja (npr. polje čvorova, naslov mape), uvoz ne uspijeva uz poruku o pogrešci.
- Djelomično valjane datoteke se ne uvoze -- sve ili ništa, kako bi se spriječile oštećene mape.

## Izvoz podataka računa

Izvezite sve svoje NeuroCanvas podatke kao jednu JSON preuzimanje iz **Postavke > Račun > Izvezi podatke**. Izvoz uključuje:

- Informacije o vašem profilu (ime, e-pošta, bio).
- Sve mape s njihovim čvorovima, vezama, pozicijama i metapodacima.
- Vaše objavljene predloške.
- Metapodatke vjerodajnica (nazivi pružatelja i oznake -- ne i kriptirani ključevi sami po sebi).

Ovo je korisno za prenosivost podataka, zahtjeve za usklađenost ili potpunu sigurnosnu kopiju računa. Ova krajnja točka je ograničena brzinom.

## Tijek rada sigurnosne kopije i vraćanja

Za redovite sigurnosne kopije:

1. **Izvozite kao JSON** periodično (tjedno ili nakon velikih sesija uređivanja).
2. Pohranite JSON datoteke na sigurno mjesto -- pohranu u oblaku, Git repozitorij ili vanjski pogon.
3. Za vraćanje, koristite gore opisani tijek uvoza.

Za sveobuhvatnu sigurnosnu kopiju, koristite gore opisani **izvoz podataka računa** za snimanje svega odjednom.

### Prijenos između platformi

Za premještanje mape s desktop aplikacije na web (ili obrnuto):

1. Na izvornoj platformi izvezite kao JSON.
2. Prenesite datoteku (pošaljite je sebi e-poštom, koristite pohranu u oblaku ili USB pogon).
3. Na ciljnoj platformi uvezite JSON datoteku.

Ovo je preporučeni tijek rada za korisnike desktopa jer se Tauri aplikacija ne sinkronizira s poslužiteljem.

### Povijest verzija

NeuroCanvas održava povijest verzija na strani poslužitelja za sinkronizirane mape. Zadnjih 20 verzija svake mape automatski se čuva, praćeno brojevima verzija sinkronizacije. Za ručnu kontrolu verzija:

1. Izvozite kao JSON na smislenim prekretnicama.
2. Imenujte datoteke s datumima ili brojevima verzija (npr. `thesis-map-2026-03-15.json`).
3. Po želji ih pohranite u Git repozitorij za razlike i potpunu vremensku liniju.
