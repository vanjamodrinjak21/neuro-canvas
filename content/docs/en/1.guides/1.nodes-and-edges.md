# Nodes and Edges

Nodes are the building blocks of your mind map. Edges are the connections between them. This guide covers everything you can do with both.

## Adding nodes

There are several ways to add a node to the canvas:

- **Keyboard shortcut**: Press `N` to activate the node tool, then click anywhere on the canvas to place a node.
- **Bottom toolbar**: Click the **Add Node** button (plus icon) in the bottom toolbar.
- **Double-click**: On an empty area of the canvas, double-click to create a node at that position.
- **AI generation**: Use Smart Expand or AI map generation to create nodes automatically (see [AI Features](/docs/guides/ai-features)).

New nodes are created with default styling and an empty text label. They appear at the click position, ready for you to type.

## Editing node content

- **Double-click** a node to enter edit mode. The node text becomes an editable field.
- Type your content. Nodes support single-line and multi-line text.
- Press `Escape` or click outside the node to confirm your edit.
- The node automatically resizes to fit its content.

## Node properties

Select a node to view and edit its properties in the right sidebar panel. Available properties:

### Content
- **Title**: The primary text displayed on the node.
- **Description**: A longer text field for notes, context, or AI-generated summaries. Not displayed on the node itself, but visible in the properties panel and used by the AI for context.

### Appearance
- **Fill color**: The node's background color. Click the color swatch to open a picker.
- **Border color**: The outline color of the node.
- **Border width**: Thickness of the node border in pixels.
- **Text color**: Color of the node's label text.
- **Font size**: Size of the label text (in pixels).
- **Font weight**: Normal or bold.
- **Shape**: Rounded rectangle (default), rectangle, circle, or diamond.

### Organization
- **Category**: A free-text label for grouping related nodes. Nodes with the same category can be visually similar or filtered together.
- **Locked**: Toggle to prevent accidental moves or edits. A locked node shows a lock icon and cannot be dragged or edited until unlocked.
- **Collapsed**: Toggle to hide all child nodes. Useful for large maps where you want to focus on high-level structure.

## Node types in the hierarchy

Nodes are visually differentiated by their position in the map hierarchy:

| Type | Visual treatment | Role |
|------|-----------------|------|
| **Root node** | Teal (#00D2BE) fill, no border | Central topic of the map. One per map. |
| **Tier-1** | Dark fill (#111113), thin border (#27272A) | Main branches off the root. Primary subtopics. |
| **Tier-2+** | Same default styling, smaller | Sub-branches and details. Depth is unlimited. |

These are default styles. You can override any node's appearance via the properties panel.

## Connecting nodes with edges

Edges represent relationships between nodes.

### Creating an edge

1. Press `C` to activate the **Connect tool**, or select it from the bottom toolbar.
2. Click on the source node (where the connection starts).
3. Drag to the target node (where the connection ends).
4. Release to create the edge.

The edge snaps to the nearest anchor point (top, right, bottom, or left) on each node, and routes automatically to minimize visual overlap.

### Edge properties

Select an edge (click on it) to configure:

- **Color**: The line color. Defaults to #3A3A42.
- **Width**: Line thickness.
- **Label**: Optional text displayed along the edge (e.g., "causes", "includes", "depends on").
- **Style**: Solid, dashed, or dotted.

### Deleting an edge

Select the edge and press `Delete` or `Backspace`. You can also right-click an edge to access the context menu and choose "Delete connection."

## Moving and resizing nodes

- **Move**: Click and drag a node to reposition it. Connected edges follow automatically.
- **Resize**: Drag the resize handle at the bottom-right corner of a selected node. Nodes have a minimum size but no maximum.
- **Snap alignment**: As you drag, smart guides appear when the node aligns horizontally or vertically with other nodes, helping you maintain clean layouts.

## Multi-select operations

You can select and operate on multiple nodes at once:

### Selecting multiple nodes
- **Shift + click**: Add or remove individual nodes from the selection.
- **Cmd/Ctrl + A**: Select all nodes on the canvas.
- **Box select**: Click and drag on an empty area of the canvas to draw a selection rectangle. All nodes within the rectangle are selected.

### Multi-select actions

When multiple nodes are selected, the **Multi-Select Bar** appears at the top of the canvas with bulk actions:

- **Delete all**: Remove all selected nodes and their edges.
- **Change color**: Apply a fill color to all selected nodes at once.
- **Align**: Align selected nodes horizontally (left, center, right) or vertically (top, middle, bottom).
- **Group move**: Drag any selected node to move the entire selection together.

## Deleting nodes

- Select one or more nodes.
- Press `Delete` or `Backspace`.
- The node is removed along with all edges connected to it.
- Child nodes are not automatically deleted -- they remain on the canvas but lose their parent connection.

::callout{type="warning"}
Deletion is undoable. Press `Cmd/Ctrl + Z` immediately to restore deleted nodes and edges.
::

## Tips for effective node organization

- **Use the root node as your thesis or central question.** Everything else branches from it.
- **Keep node labels short.** Use the description field for longer text. Short labels are easier to scan when zoomed out.
- **Use color intentionally.** Assign colors by category, priority, or status -- not decoration.
- **Collapse branches you are not actively working on.** It reduces visual noise and helps you focus.
- **Let the AI suggest connections.** After building out nodes manually, use the AI connection suggestion feature to find relationships you might have missed.
