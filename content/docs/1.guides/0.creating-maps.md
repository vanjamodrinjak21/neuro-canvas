# Creating Maps

NeuroCanvas gives you multiple ways to start a new map. Choose the approach that fits your workflow.

## Creating from the dashboard

The dashboard is your starting point after signing in.

1. Click the **+ New Map** button in the top section of the dashboard.
2. NeuroCanvas creates a new map with a single root node at the center of the canvas.
3. Click the title in the top bar to name your map.

The new map opens immediately in the canvas editor. The root node is selected by default -- start typing to set your central topic, then branch out from there.

## AI-generated maps

The fastest way to build a structured map is to let the AI do it.

### From the dashboard

1. Click **AI Quick Start** on the dashboard.
2. In the dialog, describe your topic in natural language. Be as specific or broad as you like:
   - Broad: "Machine learning"
   - Specific: "Supervised learning algorithms for tabular data in Python"
   - Question-form: "What are the main causes and effects of urbanization?"
3. Configure generation options:

| Option | Values | What it controls |
|--------|--------|-----------------|
| **Depth** | Quick, Balanced, Comprehensive | Number of nodes (~15, ~25, ~40+) |
| **Style** | Concise, Detailed, Academic | Amount of explanation per node |
| **Domain** | Free text (optional) | Focuses the AI on a specific field |

4. Click **Generate**. A loading indicator appears while the AI builds the structure.
5. The generated map opens on the canvas with a root node, main branches, and sub-branches connected by edges.

### From inside a map

While working on an existing map, open the **Generate Map** dialog from the overflow menu (three dots) in the top bar. This creates a new map rather than modifying the current one.

## Using templates

Templates provide pre-built node layouts for common use cases. From the dashboard, click **Templates** to see the available options:

| Template | Structure | Best for |
|----------|-----------|----------|
| **Brainstorm** | Central idea with 4 radiating branches | Free-form ideation, creative sessions |
| **Pros & Cons** | Decision node splitting into two columns | Evaluating options, making choices |
| **Project Planning** | Phases with milestones and tasks | Sprint planning, roadmap building |
| **SWOT Analysis** | Four-quadrant grid (Strengths, Weaknesses, Opportunities, Threats) | Strategic analysis, competitive review |
| **Timeline** | Linear sequence of events | Historical events, project timelines, process flows |

Click a template to create a new map with that layout. All nodes are editable -- rename them, add children, remove what you don't need.

## Map settings

Every map has configurable properties accessible from the top bar:

- **Title**: Click the title text to edit it inline. This is how the map appears on the dashboard and in the All Maps list.
- **Description**: Optional text that provides context. Visible on map cards and used by the AI for context-aware suggestions.

Changes to map settings are auto-saved.

## Organizing maps

### Dashboard view

The dashboard shows your most recently edited maps as cards. Each card displays:

- Map title and description preview
- Node count
- Last edited timestamp
- A miniature thumbnail of the map layout

### All Maps page

For a complete list, navigate to the **All Maps** page via the sidebar. Here you can:

- **Search** maps by title using the search bar.
- **Sort** by date modified, date created, or title.
- **Delete** maps you no longer need (with confirmation dialog).

### Import and restore

To bring in a map from another device or a backup:

1. On the dashboard, click **Import**.
2. Select a `.json` file that was previously exported from NeuroCanvas.
3. The map is restored with all nodes, edges, positions, and metadata.

See the [Export & Import](/docs/guides/export-import) guide for full details on backup workflows.
