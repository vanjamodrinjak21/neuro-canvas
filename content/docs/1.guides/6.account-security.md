# Account Security

NeuroCanvas provides several tools to keep your account secure and your data portable. All security settings are in **Settings > Account**.

## Two-factor authentication (2FA)

Add a second layer of protection to your account using a TOTP authenticator app (Google Authenticator, Authy, 1Password, etc.).

### Setting up 2FA

1. Go to **Settings > Account**.
2. In the **Two-Factor Authentication** section, click **Set Up 2FA**.
3. NeuroCanvas generates a QR code and a secret key.
4. Scan the QR code with your authenticator app (or enter the secret key manually).
5. Enter the 6-digit code from your authenticator app to verify.
6. On success, NeuroCanvas displays **10 backup codes**. Save these in a safe place -- each can only be used once.

Once enabled, you will need to enter a code from your authenticator app every time you sign in with email and password.

### Using backup codes

If you lose access to your authenticator app, enter a backup code instead of the authenticator code at sign-in. Each backup code works only once. If you run out of backup codes and lose your authenticator, you will be locked out of your account.

### Disabling 2FA

1. Go to **Settings > Account**.
2. Click **Disable 2FA**.
3. Enter your password to confirm.
4. Two-factor authentication is removed from your account.

## Password management

### Changing your password

1. Go to **Settings > Account**.
2. In the **Password** section, enter your current password and your new password.
3. Click **Change Password**.

Your new password must meet minimum strength requirements (length and character variety) and must not appear in known data breach databases.

Changing your password invalidates all existing sessions on other devices. You will stay signed in on the current device.

### Breach checking

NeuroCanvas checks your password against known data breach databases when you register and when you change your password. Your actual password is never transmitted -- a privacy-preserving protocol is used to check for breaches without exposing your password.

If your password has appeared in a known data breach, you will be asked to choose a different one.

## Data export

Download all your NeuroCanvas data as a single JSON file:

1. Go to **Settings > Account**.
2. Click **Export Data**.
3. A JSON file downloads containing:
   - Your profile information (name, email, bio)
   - All your maps with nodes, edges, positions, and metadata
   - Your published templates
   - Credential metadata (provider names and labels -- encrypted keys are not included)

This is useful for backups, data portability, or compliance requests. This endpoint is rate-limited.

## Account deletion

Permanently delete your account and all associated data:

1. Go to **Settings > Account**.
2. Click **Delete Account**.
3. Type `DELETE MY ACCOUNT` in the confirmation field.
4. Click **Confirm Deletion**.

This action is irreversible. It deletes:
- Your user profile
- All maps and their version history
- All published templates
- OAuth account links
- Active sessions
- Stored credentials
- Sync state

Account deletion is rate-limited.

## Settings overview

The **Settings** page has four tabs:

| Tab | What it controls |
|-----|-----------------|
| **General** | Theme (light/dark/system), font size, reduced motion, grid, minimap, auto-save |
| **AI Providers** | Default provider, model selection, API keys, Ollama configuration |
| **Personal** | Display name, bio (280 chars), avatar URL, language |
| **Account** | 2FA, password change, data export, account deletion |
