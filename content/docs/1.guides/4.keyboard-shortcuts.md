# Keyboard Shortcuts

NeuroCanvas supports keyboard shortcuts for nearly every common action. Learning them significantly speeds up your workflow.

Press `?` or open the shortcuts modal from the help menu to see this reference inside the app.

## Navigation

| Action | Shortcut |
|--------|----------|
| Pan canvas | `Space` + drag |
| Pan (arrow keys) | `Arrow Left` / `Right` / `Up` / `Down` |
| Zoom in | `Cmd/Ctrl` + `+` |
| Zoom out | `Cmd/Ctrl` + `-` |
| Fit all nodes to screen | `F` |
| Reset to 100% zoom | `1` |

## Tools

| Action | Shortcut |
|--------|----------|
| Select tool | `V` |
| Pan (hand) tool | `H` |
| Add node tool | `N` |
| Connect tool | `C` |
| Templates panel | `T` |

## Editing

| Action | Shortcut |
|--------|----------|
| Edit node text | Double-click node |
| Delete selected | `Delete` or `Backspace` |
| Duplicate selected | `Cmd/Ctrl` + `D` |
| Select all | `Cmd/Ctrl` + `A` |
| Undo | `Cmd/Ctrl` + `Z` |
| Redo | `Cmd/Ctrl` + `Shift` + `Z` |

## Graph and Links

| Action | Shortcut |
|--------|----------|
| Open graph view | `Cmd/Ctrl` + `G` |
| Toggle backlinks panel | `Cmd/Ctrl` + `B` |
| Toggle semantic field | `S` |

## AI Features

| Action | Shortcut |
|--------|----------|
| Smart expand (AI suggestions) | `Cmd/Ctrl` + `E` |
| Accept top suggestion | `Cmd/Ctrl` + `Enter` |

## Mouse and trackpad

| Action | Input |
|--------|-------|
| Select node | Click |
| Multi-select | `Shift` + click |
| Box select | Click + drag on empty canvas |
| Pan canvas | Middle mouse button + drag |
| Zoom | Scroll wheel |
| Context menu | Right-click |
| Move node | Click + drag on node |
| Create edge | Connect tool (`C`) + drag between nodes |

## Touch gestures (mobile / trackpad)

| Action | Gesture |
|--------|---------|
| Pan | Two-finger drag |
| Zoom | Pinch in / out |
| Select | Tap |
| Edit node | Double-tap |
| Context menu | Long press |

## Tips

- **Tool persistence**: After pressing `N` (node tool) or `C` (connect tool), the tool stays active until you press `V` (select) or `Escape`. This lets you place multiple nodes or draw multiple connections in a row without re-activating the tool.
- **Quick edits**: The fastest way to add and label a node is: `N` (activate node tool), click (place node), start typing immediately (the node enters edit mode on creation).
- **Keyboard-driven exploration**: Use `Cmd/Ctrl + G` to open the graph view for a structural overview, then `F` to fit everything to screen when you return to the canvas.
