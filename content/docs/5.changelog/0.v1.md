# v1.0.0 — Initial Release

Released: 2026

NeuroCanvas v1.0.0 is the first public release of the AI-powered mind mapping platform. This version introduces the core canvas experience, AI integration, semantic visualization, and multi-platform support.

## Infinite Canvas

- Infinite canvas with smooth pan and zoom (trackpad, mouse wheel, and pinch gesture support)
- Canvas tools: select, pan, create node, and connect
- Grid overlay with optional snap-to-grid alignment
- Level-of-detail rendering: full node detail, simplified, and dot representations based on zoom level
- Minimap with region detection and click-to-navigate
- Viewport compass indicators showing off-screen node clusters
- Keyboard shortcuts for navigation, editing, and tool switching
- Multi-select with bulk operations (move, delete, style changes)
- Node hover preview for quick inspection

## Node System

- Four node types: text, image, icon, and note
- Four node shapes: rounded, rectangle, circle, and diamond
- Full style customization: fill color, border color, border width, text color, font size, font weight, shadow, and glow
- Hierarchical parent-child relationships with collapsible subtrees
- Node locking to prevent accidental edits
- Root node designation for main topics (Petronas teal accent fill)
- Rich metadata: AI-generated descriptions, categories, keywords, and complexity ratings
- Drag-to-reposition with smooth animation

## Edge System

- Three edge types: bezier, straight, and orthogonal
- Four arrow styles: none, arrow, dot, and diamond
- Edge labels for relationship descriptions
- Customizable stroke color, width, and dash patterns
- Animated edges for visual emphasis
- AI-annotated edges with relationship type, confidence score, and reasoning

## AI Integration

- Server-side AI proxy supporting five providers: OpenAI, Anthropic, OpenRouter, Ollama, and custom OpenAI-compatible endpoints
- Smart Expand: generate related concept nodes from any existing node
- AI-powered map generation from natural language prompts with rich descriptions and hierarchical structures
- AI template generation: create reusable template structures from a topic description
- AI template adaptation: reshape an existing template for a new subject
- Node expansion suggestions with typed connections
- Auto-generated node descriptions (summary, details, keywords)
- Streaming AI responses via Server-Sent Events for real-time feedback
- AI response caching in Redis to avoid redundant API calls
- Persistent AI user memory for learning preferences over time
- Connection test endpoint for verifying provider credentials
- Configurable model, temperature, and token limits per provider
- Batch optimization and retry strategies for resilient AI operations
- Semantic cache for deduplicating similar requests

## Semantic Field Visualization

- Real-time semantic embeddings generated via transformers.js (browser) or native ONNX (Tauri desktop)
- Semantic field layer rendering concept clusters as visual regions on the canvas
- Cosine similarity computation between node embeddings
- Server-side embedding storage in Redis with 7-day TTL
- Embedding retrieval by map and node ID
- Semantic search across node content

## Template Gallery

- Browsable template gallery with category filtering (Education, Business, Creative, Planning, Research)
- Full-text search across template titles, descriptions, and tags
- Sort by popularity, newest, or alphabetical
- Template detail pages with structure preview, node count, and hierarchy levels
- One-click "Use This Template" to create a new map from any template
- Publish any map as a reusable template with title, description, category, and tags
- AI-powered template generation from topic descriptions
- AI-powered template adaptation to new subjects
- Usage tracking and popularity ranking
- Support for both system templates and user-published templates

## Guest Mode

- Try NeuroCanvas without creating an account via "Try for Free"
- Single-map guest session on the full infinite canvas
- Interactive 4-step onboarding walkthrough (add node, connect ideas, edit content, explore toolbar)
- Quick-start template picker for guest users (Brainstorm, Pros & Cons, Project Plan)
- Upgrade modal prompting sign-up when guests attempt restricted features
- Feature gating: AI, export, sharing, sync, additional maps, and account settings require an account

## Authentication

- Google OAuth, GitHub OAuth, email magic links, and email/password credentials
- Secure password hashing and breach checking via HaveIBeenPwned
- JWT session management with automatic expiry
- Password change invalidates existing sessions
- Custom auth pages: sign in, sign up, error, email verification, and OAuth redirect
- Rate limiting on all sensitive authentication endpoints
- Email enumeration prevention on registration

## Two-Factor Authentication

- TOTP-based two-factor authentication
- QR code generation for authenticator app setup
- One-time backup codes for account recovery
- TOTP verification required at sign-in when enabled
- Password re-authentication required to disable 2FA

## Account Management

- User profile management (name, bio up to 280 characters, avatar URL)
- Language selection (English, Croatian, German, French, Spanish)
- Password change with current password verification and HIBP checking
- Full account data export as JSON (profile, maps, templates, credential metadata)
- Account deletion with explicit "DELETE MY ACCOUNT" confirmation
- Cascading deletion of all associated data (sessions, OAuth accounts, credentials, maps, templates, sync state)

## Map Sharing

- Share maps via generated link tokens
- Public read-only access without sign-in for share link recipients
- Optional expiration dates on share links
- Revocable share links

## Sync Engine

- Push/pull sync protocol with optimistic concurrency control
- Checksum integrity verification
- Distributed locking to prevent concurrent write conflicts
- Version history with automatic pruning
- Soft-delete propagation across devices
- Real-time sync notifications via Server-Sent Events
- Caching layer with database fallback
- Rate limiting on sync operations
- Device ID tracking for multi-device disambiguation
- Bulk push for efficient multi-map sync

## Credential Vault

- Server-side encryption for API keys
- One credential per provider+label combination
- Support for OpenAI, Anthropic, OpenRouter, Ollama, and custom providers
- Credential metadata exposed without secrets

## Security

- CSRF protection on all state-changing endpoints
- SSRF prevention for outbound URL validation
- Input validation on all API endpoints
- Password breach checking
- Rate limiting across all sensitive endpoints

## Theme System

- Dark and light themes with system preference detection
- Petronas teal accent color (#00D2BE) throughout the interface
- CSS class-based theme switching (`.light` / `.dark` on `<html>`)
- Automatic theme persistence in user preferences

## Responsive Web Design

- Desktop-first layout with full sidebar navigation
- Mobile web support (390px target, 768px breakpoint)
- Mobile tab bar replacing the sidebar on small screens
- Responsive canvas editor with floating toolbars
- Touch-optimized controls for mobile interactions

## Settings

- Four-tab settings page: General, AI Providers, Personal, Account
- General: theme, font size, reduced motion, grid, minimap, auto-save interval
- AI Providers: default provider selection, model dropdown, API key management, Ollama toggle
- Personal: name, bio, avatar, language selection, member since date
- Account: 2FA setup/disable, password change, data export, account deletion

## Database

- PostgreSQL database with Prisma ORM
- Map version history with automatic pruning
- Map sharing with token-based access and optional expiration
- Template system with categories, tags, and usage tracking
- Encrypted credential vault for API keys

## Auto-Save

- Real-time auto-save to IndexedDB (local) and server (when authenticated)
- Debounced save to minimize write frequency
- Checksum comparison to skip no-op saves

## Export

- Export maps to JSON format (full data with nodes, edges, metadata)
- Export canvas to PNG image
- Export to Markdown (hierarchical text outline)
- Full account data export (profile, maps, templates)

## Infrastructure

- Nuxt 3 with SPA mode
- Redis for caching, rate limiting, distributed locks, pub/sub, and streams
- Docker support for self-hosting
- Health check endpoint at `/api/health`
- Node.js 20+ runtime requirement

## Desktop App (Beta)

- Tauri 2 desktop shell for macOS, Windows, and Linux
- Native ONNX runtime for local ML inference (embedding generation without browser overhead)
- Bypasses authentication in desktop mode (`__TAURI__` window detection)
- Native window management and system tray integration

## Mobile App (Beta)

- Capacitor wrapper for iOS and Android
- Shared web codebase with native shell
- Mobile-specific UI adaptations
